Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Marketplace de Entregas com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace global do ramo de recomendações de restaurantes.
O resultado final do projeto foi um painel dashboard hospedado em ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.
